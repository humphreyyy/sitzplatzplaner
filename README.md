# SeatPlaner Dittmann

Ein interaktives Tool zur Sitzplatzverwaltung und Wochenplanung für Dittmann+Ingenieure.

## Funktionen

- **Wochenplan**: 
    - Übersichtliche Darstellung der Sitzplatzbelegung pro Tag.
    - Manuelle Zuweisung von Studenten zu Sitzplätzen.
    - Auto-Zuteilungs-Funktion basierend auf Anwesenheitstagen.
    - PDF-Export und Druckansicht.
- **Studenten-Verwaltung**:
    - Hinzufügen und Entfernen von Studenten.
    - Festlegen der Anwesenheitstage (Mo-Fr).
- **Layout-Editor**:
    - Erstellen und Bearbeiten von Räumen und Sitzplätzen.
    - Drag & Drop Funktionalität.
    - Konfiguration von Sitzplatz-Eigenschaften (2 Monitore, Fensterplatz, Stehschreibtisch).

## Technologien

- React
- Vite
- Tailwind CSS
- Lucide React Icons

## Installation & Start

1.  Abhängigkeiten installieren:
    ```bash
    npm install
    ```

2.  Entwicklungsserver starten:
    ```bash
    npm run dev
    ```

3.  Build für Produktion:
    ```bash
    npm run build
    ```
