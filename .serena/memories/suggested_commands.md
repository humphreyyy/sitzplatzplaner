# Suggested Commands

## Development
- `npm install`: Install dependencies.
- `npm run dev`: Start the Vite development server (web version).
- `npm run electron:dev`: Start the Electron development version (concurrently runs Vite and Electron).

## Build
- `npm run build`: Build the web application for production.
- `npm run electron:build`: Build the Electron application for production.

## Quality Assurance
- `npm run lint`: Run ESLint to check for code quality issues.
- `npm run preview`: Preview the production build locally.

## Utility
- `npm start`: Runs `node server.js` (likely a simple backend or utility server, check `server.js` if needed).
